## 概要

AI Dinner Planner は、登録済みのメニュー情報をもとに夕食候補や材料リストを提案する
Web アプリケーションです。FastAPI の Web app と Microsoft Foundry Hosted Agent を組み合わせ、
Azure Container Apps、Cosmos DB、Azure Container Registry へデプロイする構成です。

詳細な仕様やデプロイ手順は [docs/](docs/) を参照してください。

* 仕様: [docs/spec.md](docs/spec.md)
* デプロイ手順: [docs/deployment.md](docs/deployment.md)
